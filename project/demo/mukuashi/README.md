## 毕业设计:
使用十组动画粒子做背景定时切换任务（21s执行一次新的iframe资源，循环播放），主页面添加一些项目demo演讲配合酷炫背景。10组动画预览链接：/mukuashi/animate/animate1.html (从0到10类比替换)

## 技术栈:
弹窗组件：http://layer.layui.com

动画库：https://threejs.org

jQuery等
