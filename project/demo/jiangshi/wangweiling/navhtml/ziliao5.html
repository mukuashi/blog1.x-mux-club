    课程名称： 《沟通系列——知人知心的沟通艺术》

     

    课程主讲： 王维玲

    课程对象：

    n  管理者/一线员工

    课程收益：

    n  了解沟通重要性，明确沟通要素、沟通原则、同时明确沟通的真谛

    n  掌握沟通技巧，明确沟通障碍影响因素。

    n  提高与上司的沟通质量 ，改善与下属的沟通方法

    n  通过沟通改善与其它部门的工作关系，掌握沟通理念、方法和技能，实现高效管理

     

    授课方式：

    n  讲师讲述

    n  案例精举

    n  情境演练

    n  小组讨论

    n  视频与FLASH呈现

    n  团队游戏

    n  形体训练

    n   

    课时安排：1-2天（6小时/天）

    课程大纲：

    第一模块：沟通重要性解读-----------格局决定布局

    第二模块：知人知心沟通内涵-----------布局决定结局

    第四模块：沟通艺术之解析 ----------万变不离其中
